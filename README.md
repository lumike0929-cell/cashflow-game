# 现金流自由棋

一个独立的单人财商桌游原型，灵感来自现金流类桌游的学习目标，但不是任何官方游戏的复刻。

## 玩法

1. 打开 `index.html`。
2. 选择职业。
3. 掷骰在棋盘上移动。
4. 遇到机会卡时，判断是否买入资产。
5. 遇到市场卡时，决定是否卖出资产或接受市场变化。
6. 每次经过月结日，会结算工资、被动收入、生活支出和贷款月供。
7. 当被动收入大于或等于总支出时获胜。

## 文件

- `index.html`：页面结构
- `styles.css`：视觉设计和响应式布局
- `game.js`：职业、卡牌、棋盘、现金流规则和存档逻辑
